## 笔记介绍

本笔记的任务是解读深度学习实践/面试过程中可能会用到的知识点，内容通俗易懂，入门、实习和校招轻松搞定。涵盖深度学习八股文和常用算法、模型，包括深度学习基础知识，前向传播与反向传播、神经网络、卷积神经网络CNN、循环神经网络RNN、LSTM、GRU、Bi-RNN、Transformer、GPT、BERT、T5、AE、VAE、GAN、模型的训练、优化、部署细节等。
